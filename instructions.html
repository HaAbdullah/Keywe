How to use KeyWe:

  1. Enter Code Word: Enter your unique "Code Word" in the provided text input field.
  2. Enter Websites: Below the "Code Word" input, enter the URLs of the websites you want to associate with the code word. Each URL should be entered on a new line.
  3. Save: Click the "Save" button to save your code word and associated websites.
  4. Reload: Reload your webpage to apply your new code word to chrome.
  5. Open Websites: To open the saved websites, press the backtick key (`) on your keyboard, enter the code word, and press Enter. The associated websites will open in new tabs. Note that no prompt will appear to enter the keyword. You must simply press (`) and follow it up with your codeword.
  6. Note: If one or more websites do not open from your saved sites, please allow popups and redirects through your chrome settings (Settings -> Privacy and security -> Site Settings -> Pop-ups and redirects.).

Enjoy using KeyWe and have a great browsing experience!
